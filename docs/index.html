LP8

Use Pico8 graphics and maps from Love2d

Download .zip File

Run main.lua inside Love2d for example usage.

Features:

  • Make love2d image from any sprite, with any width/height (default 8x8)
  • Get sprite flags for individual sprites, or just get all sprite flags
  • Get all map data pointing to indexes in the spritesheet
  • Create your own actual Love2d fonts using pico8 sprites!
